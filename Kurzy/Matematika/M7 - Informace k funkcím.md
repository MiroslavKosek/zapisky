## Binární relace
![[Binární relace.jpg]]
Libovolná podmnožina kartézského součinu těch dvou množin

| A\B | 1      | 2      |
| --- | ------ | ------ |
| a   | (a, 1) | (a, 2) |
| b   | (b, 1) | (b, 2) |
| c   | (c, 1) | (c, 2) |
Kartézský součin - A x B
A ... n prvků -> $2^{n}$ podmnožin
___
A ... m prvků
B ... n prvků
$|A*B|=m*n$
$|BR|=2^{m*n}=(2^{m})^{n}=(2^{n})^{m}$
